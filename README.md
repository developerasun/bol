## installation

install rust with asdf.

```sh
asdf plugin add rust
asdf list all rust
asdf install rust stable
```

refresh asdf paths.

```sh
asdf reshim
```

anchor versions in `.tool-versions`.

```
rust stable
rustc 1.93.1
rustup 1.28.2
cargo 1.93.1
```
